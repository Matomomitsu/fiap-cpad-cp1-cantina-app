# Cantina FIAP - CP1

Base do projeto em React Native + Expo para o Checkpoint 1 de Cross-Platform Application Development.

## Como rodar

```bash
npm install
npx expo start
```

## Estrutura inicial

- `app/`: rotas com Expo Router
- `components/`: componentes base reutilizaveis
- `data/`: definicoes do fluxo do app
- `assets/`: imagens e icones
- `styles/`: tema visual compartilhado
